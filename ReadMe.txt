1. If NodeJs is not installed
	Install from "https://nodejs.org/en/"
2. Open command prompt and cd into the Netchex Folder with index.js inside
	run "npm install"
	ex. "C:\User\Desktop\Netchex\Netchex>npm install"
3. Run "node index.js" 
	ex. "C:\User\Desktop\Netchex\Netchex>node index.js"
4. Run Netchex_api from visual studio
5. Open internet Browser to "LocalHost:8080"

1. Install NodeJs from "https://nodejs.org/en/"
2. In Netchex folder run "npm install" from command line
3. Run Netchex_api from visual studio
4. Open Node.js command prompt and cd into the Netchex Folder with index.js inside
5. Run "node index.js" from Node.js command prompt
	ex. "C:\User\Desktop\Netchex\Netchex>node index.js"
6. Open internet Browser to "LocalHost:8080"
